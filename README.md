# path_

A calm platform to collect, curate, and share creative AI work.

## The Problem

Creative AI work is scattered across Twitter threads, Discord servers, and GitHub repos. There's no dedicated space to share, discover, and reproduce AI experiments with proper attribution and context.

## The Solution

path_ is a minimalist platform for sharing two types of creative AI content:

- **One-shots** - A single prompt + context + model parameters with attached outputs. Fully reproducible.
- **Multi-shots** - Creative code repos and projects (GitHub, HuggingFace, ComfyUI, etc.)

## Features

- **Share** - Post one-shots with full prompt/context/model details, or link multi-shot repos
- **Discover** - Browse trending, recent, and top-rated shots
- **Engage** - Upvote quality work, bookmark for later, track views
- **Reproduce** - Every one-shot captures model, version, and parameters for reproducibility
- **Credit** - Built-in attribution system for remixes and derivatives

## Design Principles

- **Quality over quantity** - Upvotes surface the best work naturally
- **Individual merit** - Each shot stands on its own
- **Clean, focused UI** - The creative work is front and center
- **Meaningful metrics** - Views and upvotes reflect genuine interest

## Tech Stack

- Next.js (App Router)
- TypeScript
- Supabase (auth, database, storage)
- shadcn/ui
- OpenAI (auto title/description generation)
